# Автономне Обслуговування Системи

Цей процес автоматично перевіряє код, запускає тести та готує звіт без участі людини.

## 1. Перевірка здоров'я коду

```bash
python3 scripts/code_health_check.py
```

## 2. Запуск локальних тестів (Backend)

```bash
cd apps/backend && pytest tests/ --maxfail=1 --disable-warnings -v
```

## 3. Перевірка статусів Docker контейнерів (Local)

```bash
docker ps --format "table {{.Names}}\t{{.Status}}" | grep predator
```

## 4. Створення звіту

```bash
echo "Automated Check Completed at $(date)" >> automation_log.txt
```
