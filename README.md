## Quick Start (Phase 1)

### 1. Requirements

* Docker
* Kubernetes Cluster (K3s/Kind)
* Helm 3
* ArgoCD

### 2. Build Images

```bash
chmod +x start_predator.sh
./start_predator.sh
```

### 3. Deploy

```bash
kubectl create namespace predator-analytics
helm install predator ./helm/predator-analytics -n predator-analytics
```

### 4. Observe

* **ArgoCD:** `kubectl port-forward svc/argocd-server -n argocd 8080:443`
* **MCP Router:** Port 8080
* **RTB Engine:** Port 8081
