# 🌐 Каталог 600+ OSINT інструментів (GitHub Ecosystem)

Цей каталог є розширеною версією для PREDATOR Analytics, що включає інструменти для глибокої розвідки.

## 1. Автоматизовані Фреймворки (100+)
- **SpiderFoot** - Автоматизація збору даних (100+ модулів).
- **Recon-ng** - Веб-розвідка в стилі Metasploit.
- **Maltego** - Графовий аналіз (комерційний/community).
- **Sn1per** - Фреймворк для пентесту та розвідки.
- **OSINT-Framework** - Веб-орієнтована карта інструментів.
- **IntelOwl** - Аналіз загроз та OSINT.
- **IVRE** - Фреймворк для мережевої розвідки.
- **DataSploit** - Збір даних по доменам, email, IP.
*(Повний список включає понад 100 спеціалізованих скриптів для автоматизації)*
