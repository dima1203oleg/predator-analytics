# PREDATOR ANALYTICS: COGNITIVE OPERATING SYSTEM (v70.0)

## 1. Архітектурна концепція (Cognitive Operating Environment)
Веб-інтерфейс є єдиною когнітивною операційною системою. Всі модулі працюють через централізовану шину подій:
`Frontend -> UI Event Bus -> API Gateway -> [AI | ETL | Graph DB | Search | Vector DB | Analytics] -> DeepSeek-R1 (Ollama)`

## 2. Два незалежні інтерфейси
### 2.1 Інтерфейс адміністратора (Mission Control)
Призначений для повного управління платформою. Містить:
- Моніторинг сервісів (CPU, RAM, GPU, VRAM, Ollama, DeepSeek-R1, Redpanda, DBs, WebSocket-з'єднання).
- Kubernetes / Docker Dashboards, ArgoCD, Helm, GitOps.
- Керування користувачами, ролями, журнал аудиту.
- MLflow, AutoML, генерація датасетів.
- Керування парсерами, інтеграціями, Chaos Testing, Backups.

### 2.2 Інтерфейс клієнта (Business Level)
Клієнт не має доступу до системної інфраструктури. Доступні модулі:
- AI Copilot, Чат, Голосовий режим.
- Пошук, Графові зв’язки, Карта ризиків.
- Дашборди, Аналітика, Власні звіти/документи.
- Імпорт файлів, Сценарії, Сповіщення.

## 3. Центральний 3D AI-аватар
Розташований у центрі інтерфейсу.
- **Стек**: Three.js, React Three Fiber, three-vrm.
- **Вимоги**: підтримка GLB/VRM, wireframe-режим, кіберпанк-ефекти, мікроанімації (очі, моргання, поворот голови, емоції).
- **Режими**: очікування, мислення, відповідь.

## 4. Повноцінний голосовий інтерфейс
Повний цикл: `Мікрофон -> STT -> DeepSeek-R1 -> TTS -> 3D Avatar`.
- **Функціонал**: українська/англійська мови, потокове мовлення, переривання відповіді, push-to-talk, hands-free.

## 5. Інтеграція з DeepSeek-R1 (Когнітивне Ядро)
Використовується для побудови планів, аналізу документів, генерації SQL / Cypher / OpenSearch DSL, побудови RAG, пояснення висновків, роботи з MCP Tools, управління агентами.

## 6. Підтримка MCP Tool Calling
Інтерфейс підтримує виклик інструментів через MCP: PostgreSQL, Neo4j, Qdrant, OpenSearch, MinIO, Redpanda, Excel Import, ETL, AutoML, Dataset Generator. Усі виклики журналюються з перевіркою прав (RBAC).

## 7. DOM Intelligence Engine
Підсистема для аудиту інтерфейсу:
- Будує карту DOM, виявляє неактивні кнопки, перевіряє зв’язок з API.
- Контролює WebSocket-потоки, фіксує JS-помилки.
- Перевіряє актуальність даних та час рендерингу.

## 8. Реальний режим мислення AI (Safe Log)
Окрема панель показує: активних агентів, інструменти, проміжні етапи, прогрес, джерела, стан обробки, час виконання. Внутрішні "сирі" міркування приховані, виводиться лише безпечний журнал кроків.

## 9. Інтерактивний граф знань (Neo4j)
Підтримує масштабування, фільтрацію, групування, часову шкалу, підсвічування маршрутів, пошук найкоротших шляхів, кластеризацію, експорт.

## 10. Підтримка Excel-реєстру митних декларацій
Автоматичний конвеєр при завантаженні Excel:
1. Перевірка структури та валідація.
2. Збереження оригіналу в MinIO -> запуск ETL.
3. Запис у PostgreSQL -> публікація подій у Redpanda.
4. Оновлення ClickHouse, Neo4j.
5. Генерація ембедингів у Qdrant, індексація в OpenSearch.
6. Перевірка відображення в UI та підтвердження пошуку AI Copilot'ом.

## 11. Режим «Mission Control»
Єдина панель адміністратора для моніторингу хардварних (CPU/GPU) та софтверних (Celery, Redpanda, DBs, Ollama) метрик.

## 12. Самовдосконалення
Модуль для безперервного розвитку: аналіз продуктивності, пошук вузьких місць, авто-пропозиції, запуск тестів, оцінка впливу, можливість ручного затвердження змін.

## 13. Рекомендований технологічний стек
- **Frontend**: React 19, Next.js 15, TypeScript, Vite, Tailwind CSS, Framer Motion.
- **3D**: Three.js, React Three Fiber, three-vrm, WebGL.
- **Візуалізація**: Cytoscape.js, D3.js, react-force-graph.
- **Стан**: Zustand / Redux Toolkit + TanStack Query.
- **Комунікація**: REST API, WebSocket, SSE.
- **Голос**: Whisper STT, Piper TTS.
- **AI**: Ollama + DeepSeek-R1.
- **Аналітика / Сховища**: PostgreSQL, ClickHouse, Neo4j, Qdrant, OpenSearch, Redis, MinIO, Redpanda.
- **Безпека / Моніторинг**: Keycloak, RBAC, JWT, mTLS, Prometheus, Grafana, Loki.
