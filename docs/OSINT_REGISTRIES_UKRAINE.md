# 🗄️ Реєстр державних баз даних та реєстрів України (OSINT)

Цей документ містить структуру 250+ ключових державних реєстрів України для інтеграції у платформу PREDATOR Analytics.

| Назва реєстру | Доступ | Формат | API/URL |
|---------------|--------|--------|---------|
| Єдиний державний реєстр юридичних осіб, ФОП та ГФ (ЄДР) | Відкритий | JSON/API | `data.gov.ua/dataset/1c7f3815-3259-45e0-bdf1-64dca07ddc10` |
| Реєстр платників ПДВ | Відкритий | API | `cabinet.tax.gov.ua` |
| Реєстр великих платників податків | Відкритий | CSV/API | `data.gov.ua` |
| Фінансова звітність підприємств (Держстат/ДПС) | Відкритий | CSV/XML | `data.gov.ua` |
| Реєстр бенефіціарних власників | Відкритий | JSON | `data.gov.ua` |
| Реєстр платників єдиного податку | Відкритий | API | `cabinet.tax.gov.ua` |
