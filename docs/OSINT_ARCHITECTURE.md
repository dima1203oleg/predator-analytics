# Архітектура OSINT-двигуна (Рівень Palantir)

## 1. Рівень джерел даних (Data Sources)
- Державні реєстри України (250+)
- OSINT джерела (соцмережі, медіа, даркнет)
- Комерційні датасети

## 2. Рівень інгестії (Ingestion)
- Скрейпери (Scrapy, BeautifulSoup)
- API конектори
- Kafka для потокової обробки
- ETL пайплайни (Airflow, Spark)

## 3. Зберігання (Storage)
- PostgreSQL (реляційні дані)
- OpenSearch (тексти, документи)
- Neo4j (графи)
- MinIO (файли, медіа)
- Qdrant (вектори)
- ClickHouse (аналітика)

## 4. Граф знань (Knowledge Graph)
- Суб'єкти: Компанії, Особи, Продукти, Митні пости
- Зв'язки: Володіє, Керує, Імпортує, Транзакції

## 5. Аналітика (Analytics)
- Виявлення аномалій (Machine Learning)
- Мережевий аналіз (Centrality, Communities)
- Скоринг ризиків (CERS)

## 6. Візуалізація (Visualization)
- Графовий провідник (Cytoscape.js)
- Дашборди та таймлайни
