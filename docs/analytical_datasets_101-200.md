# Аналітичні Датасети для Predator v45 | Neural Analytics AI (101-200)

## Нове Покоління Аналітики
100 датасетів, що не повторюють попередні та атакують нові сфери:
- Демографія та тендери
- Архітектурні дозволи та політичні періоди
- Освітні субвенції та бюджетні асигнування
- Фіктивні ГО та криптовалютні потоки
- Еко-сертифікація та "нафталінові" ЄДРПОУ

---

## 101. "Благодійна маска"
**Опис:** Виявлення організацій, що імпортують "гуманітарку", яка потім з'являється на ринку.
**Суть:** Благодійність як прикриття комерції
**Поля:** Код пільги, Товар, Реєстр ГО, Подальший рух товару
**Приклад:** БФ "Світло надії" ввозить медобладнання, яке через місяць продається в інтернеті

## 102. "Тендерний геном"
**Опис:** Зіставлення імпорту компанії з виграними тендерами — чи товар імпортовано саме під тендер.
**Суть:** Монополізація через заздалегідь підготовлений імпорт
**Поля:** ЄДРПОУ, Дата тендеру, Дата імпорту, Код товару, Різниця днів
**Приклад:** Імпорт 500 комп'ютерів за тиждень до оголошення тендеру МОН

## 103. "Депутатський слід"
**Опис:** Аналіз імпорту компаній, пов'язаних з депутатами, в період їхнього каденційного терміну.
**Суть:** Політичний вплив на бізнес-потоки
**Поля:** ПІБ депутата, Пов'язані компанії, Обсяг імпорту, Період каденції
**Приклад:** Компанія родича депутата — +800% імпорту після виборів

## 104. "Криптомитниця"
**Опис:** Виявлення компаній, що імпортують IT-обладнання для майнінгу без декларування призначення.
**Суть:** Прихований майнінг-бізнес
**Поля:** Тип обладнання, Енергоспоживання, Кількість, Регіон
**Приклад:** 500 GPU імпортовано для "офісних потреб"

## 105. "Освітня прокладка"
**Опис:** Імпорт через університети та НДІ товарів, не пов'язаних з освітою.
**Суть:** Використання освітніх пільг для комерції
**Поля:** Назва закладу, Товар, Код пільги, Подальше використання
**Приклад:** Університет імпортує будматеріали "для лабораторії"

## 106. "Церковний щит"
**Опис:** Аналіз імпорту релігійних організацій на предмет нецільових товарів.
**Суть:** Релігійні пільги для бізнес-цілей
**Поля:** Релігійна організація, Товар, Обсяг, Митна вартість
**Приклад:** Церква імпортує "автобуси для паломників" в кількості 50 шт

## 107. "Вакцинна географія"
**Опис:** Медичний імпорт, що надходить в регіони без відповідної медичної інфраструктури.
**Суть:** Перерозподіл медичних товарів для інших цілей
**Поля:** Медтовар, Регіон призначення, Лікарень в регіоні, Обсяг
**Приклад:** Апарати ШВЛ їдуть у село з 200 жителями

## 108. "Культурна контрабанда"
**Опис:** Виявлення імпорту "антикваріату" або "мистецьких творів" з аномально низькою вартістю.
**Суть:** Заниження вартості культурних цінностей
**Поля:** Опис товару, Період створення, Страхова вартість, Митна вартість
**Приклад:** "Картина 18 ст." — митна вартість $50

## 109. "Спортивний допінг"
**Опис:** Імпорт спортивного обладнання федераціями, яке потім продається.
**Суть:** Державні кошти на спорт йдуть в комерцію
**Поля:** Федерація, Товар, Держфінансування, Подальший продаж
**Приклад:** Федерація тенісу імпортує 1000 ракеток — 900 на продаж

## 110. "Екологічна індульгенція"
**Опис:** Товари з "еко-сертифікатами" від сумнівних органів.
**Суть:** Фейкова еко-сертифікація для пільг
**Поля:** Сертифікат, Орган видачі, Товар, Перевірка в базах
**Приклад:** "Органічна" соя з Індії — сертифікат не існує в реєстрі
