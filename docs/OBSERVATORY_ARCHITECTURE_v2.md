# PREDATOR Analytics: THE OBSERVATORY (Spatial Computing Architecture)

## 1. Концептуальний фундамент
- **Парадигма:** Просторові обчислення (Spatial Computing) замість плоских дашбордів. 
- **Естетика:** Military Cyber Intelligence / Premium Enterprise. Dark Glassmorphism, мінімалізм. Без надлишкового футуризму. 
- **Кольорова палітра:** Чорний, графіт, темно-синій, холодний білий. 
  - *Бірюзовий:* довірені зв'язки. 
  - *Золотий:* цінні активи. 
  - *Помаранчевий:* виявлені ризики. 
  - *Червоний:* критичні події.
- **Продуктивність:** 60-120 FPS.

## 2. Технологічний стек та екосистема
- **База:** React 18 + Next.js 14 + Tailwind CSS + shadcn/ui.
- **3D Рендеринг:** Three.js + React Three Fiber (R3F) + `@react-three/drei`.
- **Глобальний стан:** Zustand (обхід циклу React через `useFrame` та `useRef` для високочастотних оновлень позицій).
- **Підключення:** WebSocket для потокових даних.
- **SSR Обмеження:** Усі 3D-компоненти обгортаються в `next/dynamic` з `ssr: false`.

## 3. Центральне живе ядро (The Core)
- **Сутність:** Пульсуючий енергетичний кристал / багатогранник.
- **Реалізація:** Кастомні GLSL Shaders (Vertex для морфінгу через Simplex Noise + Fragment для Fresnel Effect).
- **Світіння:** "Fake Glow" (`fake-glow-material-r3f`) замість важкого `UnrealBloomPass`.
- **Взаємодія:** Колір та частота морфінгу залежать від навантаження (червоний при критичних подіях).

## 4. 3D-Граф зв'язків (Force Graph 3D)
- **Рішення:** `react-force-graph-3d` (рушій `d3-force-3d`).
- **Кастомні вузли (`nodeThreeObject`):**
  - *Компанії:* Великі напівпрозорі багатогранники з металевим блиском.
  - *Люди:* Малі сфери/тетраедри (бірюзове світіння).
  - *Документи:* Інформаційні кристали (октаедри).
  - *Контейнери:* Рухомі блоки вздовж зв'язків.
- **Зв'язки:** Енергетичні потоки (TubeGeometry). Використання напрямлених частинок (`emitParticle`) для візуалізації ризиків (червоні імпульси).

## 5. Просторова часова шкала
- Розподіл вузлів вздовж осі (Z або X) пропорційно до їхнього `timestamp`.
- Фільтрація масиву `graphData` для "подорожей у часі", після якої рушій D3 плавно перекомпоновує топологію.

## 6. Геопросторова візуалізація (3D Карта)
- **Рішення:** `deck.gl` + `MapLibre GL JS` (Base Map).
- **Інтеграція:** Interleaved mode (`MapboxOverlay` з `interleaved: true`), щоб карта існувала в єдиному WebGL-контексті з 3D об'єктами.
- **Живі маршрути:** `TripsLayer` для відображення світлових "хвостів" кораблів та контейнерів у реальному часі.

## 7. Робота з документами (ШІ + PDF)
- **Рішення:** `react-pdf-highlighter-plus` поверх Canvas (абсолютне позиціонування + Glassmorphism).
- **Потік:** PDF → FastAPI (NER екстракція) → Фронтенд відображає підсвітки (TextHighlight) за кольовим кодом.
- **Навігація:** Клік на слово (суть) в PDF → Зміна стану Zustand → 3D-камера плавно перелітає (`fly-to`) до відповідного вузла у Force Graph.

## 8. UI та Інфраструктурний моніторинг
- **Дизайн панелей:** Headless UI (`shadcn/ui`), стилізація Tailwind (напівпрозорість, розмиття `bg-black/40 backdrop-blur-xl`).
- **Left Panel:** Навігація між модулями (змінює контекст 3D-сцени).
- **Right Panel:** Когнітивний ШІ-асистент (JSON висновки).
- **Bottom Status Bar:** Телеметрія у реальному часі (CPU, VRAM, Kafka, DBs) через OpenTelemetry/WebSocket. Компоненти `Progress` відображають стан різними кольорами.

## 9. Візуалізація ШІ (Background)
- **Рішення:** Volumetric Fog або Particle Systems. Енергетичні згустки навколо графа/ядра, що реагують на навантаження ШІ-агентів. Жодних антропоморфних образів.

## 10. Правила генерації коду (Antigravity Directives)
- **Dual-Scope Strategy:** Чітке розділення DOM та Canvas. Ніяких HTML-тегів усередині `<Canvas>`.
- **Artifacts:** Кожен логічний блок (шейдер, парсер, deck.gl) створюється та тестується ізольовано.
- **Subagents:** Використання спеціалізованих субагентів для WebGL, Tailwind та Zustand.
- **Strict Typing:** Типізація через TypeScript (Pydantic V2 на бекенді → JSON Schema). Суворі типи для `NodeData` та `LinkData`.
