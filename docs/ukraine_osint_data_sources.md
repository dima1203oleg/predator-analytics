# Джерела відкритих даних України та API для OSINT

## Українські державні джерела з API / відкритими дампами

### 1. Портал відкритих даних України — data.gov.ua
**Що це:** Єдиний державний веб-портал відкритих даних з сотнями наборів.
**Де:** [data.gov.ua](https://data.gov.ua)
**Формати:** CSV, JSON, XLS(X), GeoJSON, XML, RDF.
**Використання:** Економічні, демографічні, бюджетні і транспортні набори.

### 2. Електронний кабінет ДПС – відкриті реєстри API
**Що це:** REST API для відкритої частини податкових реєстрів (з токеном).
**Підтримуються реєстри:** ПДВ, єдиного податку, страхувальників, неприбуткових.
**Формат:** JSON.
**Авторизація:** Токен у заголовку запиту (потребує регулярної регенерації через ліміти).

### 3. Державна митна служба — відкриті дані та API декларацій
**Що це:** Держмитслужба відкрила API-доступ до митних декларацій для бізнесу.
**Використання:** Автоматичне отримання декларацій, групування по HS кодах, брокерах, постах.

### 4. OpenDataBot API (ЄДР, ЄРСР, ЄРБ, АСВП, РРП)
**Що це:** Єдиний REST API доступ до численних державних реєстрів через один інтерфейс.
**Підтримує:** ЄДР, судові рішення, реєстр боржників, речових прав, АСВП.
**Формат:** JSON.

### 5. Єдиний державний реєстр судових рішень (ЄДРСР)
**Що це:** Повний реєстр судових рішень з доступом утилітарними API/форматами через data.gov.ua.
**Використання:** Аналітика юридичних ризиків, фігурантність у справах.

### 6. Портал "Стан розгляду справ" (Судова влада України)
**Що це:** Офіційний портал, що показує поточний статус судових справ (парсинг або API).

### 7. SIS NIPO — API інтелектуальної власності
**Що це:** API держсистеми інтелектуальної власності України (патенти, ТМ).
**Точка доступу:** https://sis.nipo.gov.ua/api/v1/open-data/

### 8. Реєстри Судова влада України — статус справ
**Що це:** Портал court.gov.ua дозволяє перевіряти, чи є особа стороною у справі.

### 9. Реєстр транспортних засобів
**Що це:** Відкриті дані про транспортні засоби, VIN-код, реєстрацію та власників (на data.gov.ua).

### 10. Prozorro — офіційний API держзакупівель
**Що це:** API тендерної системи api.prozorro.gov.ua.
**Використання:** Аналіз держзакупівель, корупційних схем, зв’язків контрагентів. Формат: JSON.

### 11. Є-Data / Spending.gov.ua
**Що це:** Відкриті дані про державні витрати, транзакції, портал публічних фінансів.

### 12. Статистичні дані державної статистики (Ukstat)
**Що це:** Відкриті дані про демографію, виробництво, ринки праці.

### 13. Морські та транспортні реєстри (Maritime)
**Що це:** Реєстри портів, суден, перевезень, вантажних потоків.

### 14. Санкції, PEP та комплаєнс-списки
**Що це:** Державні санкційні реєстри (РНБО) і реєстри політично значущих осіб.

### 15. Медіа та Social API
**Що це:** Інтеграція з Telegram, RSS-стрічками новин та X/Twitter API.

## Автоматизація завантаження даних (Приклади)

### 1. data.gov.ua / CKAN API
```bash
curl -O "https://data.gov.ua/api/3/action/package_show?id={dataset_id}"
```

### 2. OpenDataBot API
```python
import requests
url = "https://api.opendatabot.ua/v1/entity/edrpou/{code}"
resp = requests.get(url, params={"token": "<ваш_токен>"})
```

### 3. Електронний кабінет ДПС
```bash
curl -H "Authorization: Token <ваш_токен>" "https://cabinet.tax.gov.ua/api/registers/pdv"
```

### 4. Prozorro API
```bash
curl "https://public.api.openprocurement.org/api/2.5/tenders?offset=0&limit=50"
```
