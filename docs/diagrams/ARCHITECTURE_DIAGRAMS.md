## 3. Self-Healing Architecture

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                      SELF-HEALING ARCHITECTURE                               │
└─────────────────────────────────────────────────────────────────────────────┘

              ┌───────────────────────────────────────────┐
              │              GUARDIAN SERVICE              │
              │          (Головний контролер)              │
              └───────────────────┬───────────────────────┘
                                  │
        ┌─────────────────────────┼─────────────────────────┐
        │                         │                         │
        ▼                         ▼                         ▼
┌───────────────┐         ┌───────────────┐         ┌───────────────┐
│ HEALTH CHECK  │         │  AUTO RESTART │         │ AUTO ROLLBACK │
│               │         │               │         │               │
│ • HTTP /health│         │ • Docker      │         │ • Git revert  │
│ • DB connect  │         │ • K8s restart │         │ • ArgoCD sync │
│ • Redis ping  │         │ • Process mgr │         │ • Helm rollback│
│ • Queue depth │         │               │         │               │
└───────┬───────┘         └───────┬───────┘         └───────┬───────┘
        │                         │                         │
        │         ┌───────────────┼───────────────┐         │
        │         │               │               │         │
        ▼         ▼               ▼               ▼         ▼
        ┌─────────────────────────────────────────────────────┐
        │                    RECOVERY ACTIONS                  │
        │                                                      │
        │  1. Detect failure (Prometheus alert)                │
        │  2. Guardian triggers recovery                       │
        │  3. Restart service / Rollback deployment            │
        │  4. Verify health                                    │
        │  5. Notify admin (if recovery failed)                │
        └─────────────────────────────────────────────────────┘
```
