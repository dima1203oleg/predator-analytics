# 🦁 PREDATOR PLATFORM V30: GLOBAL STRATEGY & PREMIUM EVOLUTION
*Confidential Technical & Business Specification*

## 1. ФІЛОСОФІЯ СИСТЕМИ: "TOTAL INTELLIGENCE"
Система PREDATOR v30 перетворюється з інструменту "аналітики даних" на платформу **"інформаційної переваги"**. Вона не просто обробляє дані, вона дає владу.

### 🧠 ДВА ВЕКТОРИ ЦІННОСТІ (DUAL-PURPOSE CORE)
Система працює на єдиній базі даних, але має два абсолютно різних інтерфейси (або режими відображення) залежно від ролі клієнта.

#### 🏛️ ВЕКТОР A: "CONTROL & ENFORCE" (Для Держструктур / Силовиків)
**Ціль:** Знайти порушення, зв'язки, слабкі місця, компромат.
**Монетизація:** Продаж ліцензій на відділи, кастомні аналітичні запити, "підписка на порушника".
**Ключові Функції:**
*   **Red Flag Detector**: Автоматичне підсвічування аномалій (занижена вартість, фіктивні директори, офшорні зв'язки).
*   **Scheme Hunter**: Виявлення ланцюжків поставок, що обходять санкції або податки ("скрутки").
*   **Predictive Warrant**: Автоматична генерація доказової бази (звітів) для відкриття проваджень.
*   **Target Monitoring**: "Поставити на прослушку" — моніторинг діяльності конкретної юрособи у всіх реєстрах + Telegram.

#### 💼 ВЕКТОР B: "EARN & DOMINATE" (Для Бізнесу / Комерції)
**Ціль:** Знайти маржинальні товари, дізнатися ціни конкурентів, оптимізувати логістику, перехопити ринок.
**Монетизація:** Щомісячна підписка (SaaS), плата за звіти, доступ до історичних даних.
**Ключові Функції:**
*   **Competitor X-Ray**: Повний розклад імпорту конкурента (об'єми, ціни, постачальники, клієнти).
*   **Market Gap Finder**: Пошук товарів, де попит перевищує пропозицію (аналіз трендів).
*   **Price Benchmark**: "Чи не переплачую я?" — порівняння своїх закупівельних цін з середньоринковими.
*   **Lead Generator**: Список потенційних клієнтів, які купують у ваших конкурентів.

## 3. MULTI-SOURCE INGESTION ENGINE (ВСЕЯДНІСТЬ)
Система поглинає дані у будь-якому форматі. Це критично, бо інсайти ховаються не тільки в Excel.

### 3.1 Supported Inputs (Вже Реалізовано)
*   ✅ **Excel / CSV**: Митні реєстри, транзакції (1GB+ підтримка).
*   ✅ **API**: Prozorro, E-Data, OpenSanctions.
*   ✅ **Web Scraping**: Новини, сайти конкурентів.

### 3.2 Advanced Ingestion (Нове в v30)
*   🎥 **Video/Audio Intelligence**:
    *   Моніторинг YouTube/TikTok/Telegram каналів.
    *   **Whisper AI Parsing**: Транскрибація голосу в текст.
    *   **Sentiment Analysis**: Про що говорять? (Паніка, інсайд, злив).
    *   **Entity Extraction**: Витягування прізвищ та назв компаній з аудіо.
*   📄 **Document OCR**: Розпізнавання сканованих накладних, інвойсів, контрактів (PDF/Images).
*   📱 **Telegram Spy**: Глибокий аналіз закритих чатів (хто з ким спілкується, теми-табу).

## 4. ТЕХНІЧНА РЕАЛІЗАЦІЯ ТА ПОКРАЩЕННЯ (ROADMAP)

### 4.1 Індивідуальні Пайплайни (Per-Source Observability)
Клієнт повинен бачити, як його файл "перетравлюється".
*   **UI**: Прогрес-бар для кожного файлу: Upload -> Virus Check -> Parse -> Enrich -> Store -> Index.
*   **Real-time**: WebSocket оновлення статусу.

### 4.2 Dataset Studio & Prototypes
"Навчи систему на моєму досвіді".
*   Клієнт завантажує свої "золоті дані" (приклад ідеальної угоди або типового порушення).
*   Натискає **"Set as Prototype"**.
*   **Predator Syntho-Forge** генерує тисячі подібних сценаріїв для навчання AI-моделей для детекції.

### 4.3 Усунення Технічних Блокерів (Immediate Fixes)
1.  **Dependency Hell**: Використання "Resilient Import" паттерну для `structlog`, `opentelemetry`, `psycopg2`.
2.  **Upload Limits**: Зняття ліміту 100MB, перехід на 1GB (Chunked Upload).
3.  **Backend Stability**: Автоматичний перезапуск воркерів при OOM (Out Of Memory).
