---
description: Автоматичний процес фіксації та відправки змін (Commit & Push)
---

// turbo-all
1. Перевірити стан репозиторію за допомогою `git status`.
2. Додати всі змінені та нові файли: `git add .`
3. Згенерувати повідомлення коміту відповідно до правил `AGENTS.md` (наприклад, `feat(core): опис змін`). Повідомлення має бути українською мовою.
4. Виконати коміт: `git commit -m "ваш_меседж" --no-verify`. (Використовуємо `--no-verify`, щоб обійти проблеми з локальним pre-commit оточенням).
5. Виконати оновлення з репозиторію для уникнення конфліктів: `git pull --rebase`.
6. Відправити зміни в основну гілку: `git push`.
7. Повідомити USER про успішне завершення.
