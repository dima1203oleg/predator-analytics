## DEFINITION OF DONE

Новий конектор вважається готовим лише після того, як:
* знайдено джерело;
* виконано профілювання;
* визначено схему;
* створено конектор;
* реалізовано інкрементальне оновлення;
* реалізовано потокове завантаження для великих наборів даних (де це застосовно);
* побудовано мапінг сутностей;
* згенеровано ETL;
* інтегровано зі сховищами;
* створено граф знань;
* створено векторні індекси;
* пройдено повний цикл тестування;
* створено документацію;
* успішно виконано CI/CD;
* конектор переведено в режим безперервного моніторингу;
* увімкнено автоматичне самовідновлення при змінах API.
