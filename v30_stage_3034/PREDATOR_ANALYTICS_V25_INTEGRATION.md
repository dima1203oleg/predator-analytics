# Інтеграція OpenSearch Dashboards Analytics (v25)

## 🚀 Огляд реалізації
Було виконано повну інтеграцію аналітичних графіків OpenSearch та метрик реального часу в інтерфейс Predator Analytics UI. Всі компоненти переведені на реальні дані без використання симуляцій.

## 🛠 Створені Компоненти

### 1. `AnalyticsDashboard`
**Шлях:** `src/components/premium/AnalyticsDashboard.tsx`
- **Призначення:** Основний аналітичний центр.
- **Функціонал:**
  - Відображення CPU, RAM, Disk I/O в реальному часі.
  - Графіки запитів та латенсі (ECharts).
  - Кільцева діаграма розподілу даних по сховищах (OpenSearch, Qdrant, MinIO, Postgres).
  - Статус здоров'я всіх сервісів.

### 2. `OpenSearchLiveWidget`
**Шлях:** `src/components/premium/OpenSearchLiveWidget.tsx`
- **Призначення:** Компактний віджет для головного дашборду.
- **Функціонал:**
  - Живі лічильники документів та індексації.
  - Міні-графік спарклайн для активності.
  - Пряме посилання на повний інтерфейс OpenSearch Dashboards (IP: 194.177.1.240).

### 3. `OpenSearch Premium Component`
**Шлях:** `src/components/premium/OpenSearch/index.tsx`
- **Оновлення:** Додано гібридний режим.
- **Режими:**
  - **Native:** Відображення локальних графіків на базі ECharts.
  - **Iframe:** Вбудовування повного інтерфейсу Kibana/OpenSearch Dashboards.

## 🔗 Інтеграції

### Головний Дашборд (`DashboardView.tsx`)
- Додано `OpenSearchLiveWidget` в секцію аналітики.
- Додано компактні віджети для Qdrant, PostgreSQL, MinIO.

### Моніторинг (`MonitoringView.tsx`)
- Вкладка **ANALYTICS** повністю оновлена.
- Замість статичних карток тепер використовується `AnalyticsDashboard`.

## ⚙️ Технічні Зміни
- **Vite Config:** Тимчасово вимкнено PWA плагін для вирішення проблеми з правами доступу (EPERM) на macOS.
- **API Service:** Перевірено та підтверджено наявність v25 ендпоінтів для отримання метрик.

## ▶️ Як запустити
Оскільки на машині діють обмеження безпеки (Sandbox/SIP), рекомендується запускати через npm стандартним чином:

```bash
cd apps/predator-analytics-ui
npm run dev
```

Система автоматично підключиться до бекенду та почне відображати реальні дані.
