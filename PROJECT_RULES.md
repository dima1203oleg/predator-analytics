# ПРАВИЛА ПРОЕКТУ PREDATOR ANALYTICS

## 1. МОВНИЙ ЗАКОН (UKRAINIAN LANGUAGE LAW)
**СТАТУС: КРИТИЧНИЙ / ОБОВ'ЯЗКОВИЙ**

1.  **Спілкування**: Асистент (Antigravity/AI) зобов'язаний спілкуватися з Користувачем **ВИКЛЮЧНО УКРАЇНСЬКОЮ МОВОЮ**.
    *   Це стосується всіх відповідей, пояснень, коментарів до коду та комітів.
    *   Англійська допускається лише в назвах змінних, функцій, бібліотек та технічних термінах, де переклад недоречний (наприклад `Docker`, `React`, `useEffect`).

2.  **Інтерфейс (UI/UX)**:
    *   Увесь інтерфейс веб-додатку (кнопки, меню, модальні вікна, повідомлення про помилки, статуси) має бути локалізований українською мовою.
    *   Формат дат: `DD.MM.YYYY` або український стандарт (наприклад, "11 січня 2026").
    *   Валюта: Гривня (₴) або USD ($) з відповідним підписом.

## 2. ТЕХНІЧНИЙ СТАНДАРТ (PYTHON 3.12 ONLY)
**СТАТУС: НЕЗМІННИЙ**
*   Єдина допустима версія Python: **3.12**.
*   Усі Docker-контейнери, віртуальні середовища та скрипти повинні використовувати виключно Python 3.12.
*   Використання інших версій (3.9, 3.10, 3.11) суворо заборонено.

## 3. МАКСИМАЛЬНА АВТОМАТИЗАЦІЯ (GOD MODE)
*   **Root/Global Access**: Система має повні права `root` та глобальний доступ до всіх компонентів.
*   **Автосхвалення**: Всі критичні виправлення, оновлення та розгортання виконуються автоматично, без зайвих запитань про дозвіл (Turbo Mode).
*   **Пріоритет стабільності**: Вирішення блокуючих проблем (диск, пам'ять, мережа) відбувається миттєво і пріоритетно.
