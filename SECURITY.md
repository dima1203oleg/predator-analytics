# Безпека — Рекомендації (Predator Analytics v22.0)

Критично: цей файл містить кроки для негайних дій у разі виявлення секретів в репозиторії.

## Дія при виявленні секретів
1. Ротація: негайно змініть/відключіть ключі/паролі на провайдері сервісів.
2. Видалення з історії Git та .gitignore:

```bash
# Видалити файл з індексу
git rm --cached predator-analytics/ua-sources/dynamic_keys.json
# Додати файл до .gitignore
echo "**/dynamic_keys.json" >> .gitignore
```

3. Повне видалення з історії (git-filter-repo) - погодьтеся команадою та виконайте резервні копії.

## Recommendations
- Використовувати HashiCorp Vault або cloud secret manager.
- Додавати secret-scan в CI перед мерджем.
