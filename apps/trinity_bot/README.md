# Predator Trinity Bot v45.0

## Overview
Trinity Bot is the "Reality Terminal" for the Predator Analytics platform. It leverages a triple-agent architecture (Gemini, Mistral, Copilot) to provide autonomous management, code generation, and infrastructure control via Telegram.

## Structure
- `bot/main.py`: Entry point and Telegram interaction logic (aiogram).
- `core/cortex.py`: The "Brain" - orchestrates Gemini (Strategist), Mistral (Coder), and Copilot (Auditor).
- `core/tools.py`: Infrastructure interaction layer (K8s, RabbitMQ, ArgoCD).
- `utils/voice.py`: Voice processing (STT/TTS) with local fallback.

## Deployment
1. **Secrets**: Configure `k8s/bot/secrets.yaml` with your Vault/ExternalSecret provider.
2. **Build**: Build the image using the provided `Dockerfile`.
3. **Apply**: `kubectl apply -f k8s/bot/`

## Features
- **Voice Commands**: Integrated STT for hands-free cluster management.
- **Triple-Agent Audit**: Every code snippet generated by Mistral is audited by Copilot and planned by Gemini.
- **FinOps Integrated**: GPU scaling and resource management with budget checks.
- **Shadow Protocol**: Secure access to classified system intelligence.
