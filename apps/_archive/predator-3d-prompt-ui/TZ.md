# ✅ Технічне Завдання (ТЗ)
на дизайн користувацького веб-інтерфейсу **PREDATOR Analytics**  
(на основі концепції **«The Hunter’s Void Throne»**)

### 1. Назва проєкту та концепція

- **Назва концепції:** The Hunter’s Void Throne (Трон Хижака в Порожнечі)
- **Суть:** Реалістичний sci-fi веб-інтерфейс, де користувач виступає в ролі командира-мисливця, а вся аналітика відбувається всередині космічної станції з хижаком як центральним символом.

### 2. Мета проєкту

- Створити сучасний, реалістичний і візуально вражаючий веб-інтерфейс.
- Досягти сильного **wow-ефекту** при першому знайомстві з продуктом.
- Зробити інтерфейс **максимально корисним** для професійних аналітиків.
- Створити унікальну візуальну мову, яка відрізняє PREDATOR Analytics від конкурентів.

### 3. Візуальна концепція

**Основна метафора:**  
Хижак (Yautja) = символ «полювання на інформацію».  
Користувач рухається космічною доріжкою між планетами-порталами, кожна з яких — окремий модуль програми.

**Ключові візуальні елементи:**

- Хижак сидить на масивному троні в центрі круглої кімнати.
- За спиною — велике вікно в космос з туманністю.
- Перед хижаком — голографічний стіл з даними.
- Космічна доріжка по колу.
- Планети-портали як навігація.
- Лазери хижака як візуалізація сканування зв’язків.

**Стиль:** Реалістичний sci-fi з кінематографічним освітленням (PBR, reflections, volumetric lighting).

### 4. Вимоги до стилю та якості

- Реалістичний sci-fi (не мультяшний).
- Висока деталізація та якість матеріалів.
- Сильний wow-ефект через атмосферу, освітлення та деталі.
- Баланс між красою і практичністю.

### 5. Структура інтерфейсу

**Основні зони:**

- Верхня глобальна панель
- Бокова навігація (модулі)
- Головна 3D-область (кімната + хижак + доріжка)
- Панель деталей / результатів

**Навігація:**

- Через **космічну доріжку** та **планети-портали** (замість класичного меню).

### 6. Вимоги до 3D-візуалізації

|Елемент              |Вимога                                 |Модель (рекомендовано)           |
|---------------------|---------------------------------------|---------------------------------|
|Базова кімната       |Реалістична sci-fi кімната             |Sci-Fi Control Room (Dyonos)     |
|Хижак                |Висока деталізація + анімація + фізика |Jungle Hunter Predator           |
|Фон космосу          |Реалістичний космос + туманність       |Deep Space Skybox + Nebula Skybox|
|Кораблі-розвідники   |Літають між планетами                  |LIGHT FIGHTER SPACESHIP          |
|Планети-портали      |Унікальний стиль для кожного модуля    |Кастомні або готові моделі планет|
|Лазери               |Реалістичний beam + volumetric lighting|—                                |
|Голографічні елементи|Прозорі екрани з reflections           |—                                |

**Анімації та фізика:**

- Ходьба хижака по доріжці
- Реалістична фізика гребня (dreadlocks)
- Анімація лазерів при скануванні
- Плавні переходи між планетами (camera animation)

### 7. Функціональність та корисність

- Лазери хижака візуально підсвічують приховані зв’язки.
- Кораблі-розвідники «збирають дані».
- Голографічні звіти можна перетягувати.
- Фізичне переміщення доріжкою дає відчуття процесу «полювання».
- Кожна планета має унікальний візуальний стиль і атмосферу.

### 8. Технічні вимоги

- **Стек:** Next.js 14+ + TypeScript + Tailwind
- **3D:** React Three Fiber + @react-three/drei + postprocessing
- **Фізика:** rapier3d або cannon-es (для гребня)
- **Моделі:** GLB з Draco compression + LOD
- **Освітлення:** HDRi + фізично коректне освітлення
- **Адаптивність:** Повна підтримка десктопу (від 1440px)

### 9. Моделі (безкоштовні)

- Sci-Fi Control Room — Dyonos
- Jungle Hunter Predator (702k)
- Deep Space Skybox
- Nebula Skybox
- Light Fighter Spaceship

### 10. Критерії якості

- Інтерфейс викликає сильний wow-ефект.
- Стиль реалістичний і професійний.
- Візуалізації допомагають краще сприймати дані.
- Код чистий, оптимізований і підтримуваний.
- Повністю відповідає концепції «The Hunter’s Void Throne».
