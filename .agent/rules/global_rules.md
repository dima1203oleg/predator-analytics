# Predator Analytics v25 Global Rules

## 1. Мовна Політика

- **Основна мова**: Українська (UA).
- Всі коментарі в коді (де це доречно), документація, звіти та повідомлення користувачу мають бути українською мовою.
- Повна локалізація UI/UX: перевіряти наявність перекладів у `uk-UA.json` та інших i18n файлах.

## 2. Технологічний Стек

- **Python**: Виключно версія **3.12**.
- Використовувати сучасні можливості Python 3.12 (наприклад, покращені generic types, f-strings).
- Кожен новий скрипт повинен мати перевірку версії або чітку вказівку в `pyproject.toml` чи `requirements.txt`.

## 3. Стандарти Розробки

- Всі системні виклики та API повинні підтримувати UTF-8.
- Логування має бути структурованим (JSON) з підтримкою кирилиці.
- При розгортанні на серверах NVIDIA завжди пріоритезувати локальні ресурси.
