# 🛡️ /azr_guard: Глобальний Контроль Якості

Цей воркфлоу активує систему перевірки відповідності проєкту Аксіомі 15 Конституції AZR.

**ОБОВ'ЯЗКОВІ ПЕРЕВІРКИ:**
1. **Мова**: Тільки Українська в інтерфейсі та коментарях.
2. **Runtime**: Тільки Python 3.12 в усіх Dockerfiles.
3. **Синхронізація**: Перевірка на дублікати (напр. AppRoutes vs AppRoutesNew).
4. **Інфраструктура**: Перевірка статусу ngrok та внутрішніх посилань.

// turbo
1. Запустити конституційну перевірку:
```bash
python3 scripts/azr_constitutional_guard.py
```

2. Якщо виявлено порушення - ПРИПИНИТИ розгортання та виправити.

3. Перевірити статус ngrok:
```bash
scripts/check_ngrok_service.sh
```

4. Корекція інтерфейсу (монолітизація):
При зміні стабільних компонентів, використовувати `monolith_mode=true` для фіксації версії.
